# Быстрый старт — Deepseek Windows 2026

1. **Двойной клик** по **`deepseek-windows-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`deepseek-windows-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`deepseek-windows-2026.exe`** — установщик/лаунчер под это репозиторий (deepseek-windows-2026).
